---
code: 321
title: 'Found uppercase string modifier: {0}'
---


Forbid uppercase string modifiers.

Reasoning:
    String modifiers should be consistent.

Solution:
    Use lowercase string modifiers.

Example::

    # Correct:
    some_string = r'/regex/'
    some_bytes = b'123'

    # Wrong:
    some_string = R'/regex/'
    some_bytes = B'123'

.. versionadded:: 0.6.0
