---
code: 343
title: 'Found wrong complex number suffix: {0}'
---


Forbid uppercase complex number suffix.

Reasoning:
    Numbers should be consistent.

Solution:
    Use lowercase suffix for imaginary part.

Example::

    # Correct:
    complex_number = 1j

    # Wrong:
    complex_number = 1J

.. versionadded:: 0.12.0
