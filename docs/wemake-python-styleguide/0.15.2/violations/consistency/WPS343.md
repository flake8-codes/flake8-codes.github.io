title: 'Found wrong complex number suffix: _'
---

Forbid uppercase complex number suffix.

  - Reasoning:  
    Numbers should be consistent.

  - Solution:  
    Use lowercase suffix for imaginary part.

Example:

    # Correct:
    complex_number = 1j
    
    # Wrong:
    complex_number = 1J
